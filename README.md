# 🌍 Yatrika – Travel Place Recommendation System

Yatrika is an AI-powered travel place recommendation web application built to help users discover amazing destinations across India and around the world.

The platform showcases:

* Trending destinations
* Hidden gems
* Daily travel picks
* Recently popular places
* Beautiful interactive UI
* Destination categories with ratings and crowd indicators

## ✨ Features

* Modern responsive travel UI
* 100+ travel destinations
* Favorite/like system with animation
* Smart category filtering
* Indian & international destinations
* Beautiful image-based cards
* AI-curated recommendations

## 🛠️ Tech Stack

### Frontend

* React
